This project is a web-based tiffin subscription platform that connects students with nearby home-style food providers in a 
2 km radius. It allows students to discover providers, browse subscription plans, view detailed day-wise menus, and place 
lunch or dinner subscriptions online. Providers can register their tiffin service, publish multiple plans, configure daily items 
for each plan, and manage incoming subscription orders from a dashboard. The system tracks order status end-to-end 
(placed, accepted, cancelled, delivered) so both student and provider have clear visibility. The main goal is to replace manual 
WhatsApp/phone-based tiffin booking with a structured, transparent system that reduces confusion about which plan was 
booked, what food is included on each day, and the overall subscription duration and cost. The project targets hostel 
students and working professionals who need reliable, affordable recurring meals near their location.
